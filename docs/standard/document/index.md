# システム関連図

システム関連図は、関連するアクターや外部システムとの関連を示す図です。例として採用管理システムを開発する初期フェーズでモデリングを行ったのが次の図です。

```puml
@startuml
title : プロジェクト名 レベル0
node "管理会社" as company
actor/ 応募者 as aaaa
aaaa -> company :精算書
note top of aaaa : This is an example.


@enduml
```

```puml
@startsalt
{+
サインイン
 UserId:   | "      "
 Password: | "***   "
 [ ] サインインしたままにする
[キャンセル] | [サインイン]
}


@endsalt
```
